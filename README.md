# UIDynamicList

# Unity UI System: Расширяемые Панели с Автопрокруткой

## Обзор проекта
Профессиональная система UI для Unity, реализующая интерактивный список с расширяемыми панелями, автоматической прокруткой к выделенным элементам и кастомными шейдерами градиентов. Система идеально подходит для создания рейтингов, каталогов, меню настроек и других сложных интерфейсов.

## Ключевые возможности

### Умная навигация
- **Двойное управление** - мышью и клавиатурой (стрелки вверх/вниз/вправо/влево)
- **Автоматическая прокрутка** - ScrollRect плавно скроллится к текущему выделенному элементу
- **Сеточная навигация** - поддержка многоколоночных макетов

### Интерактивные элементы
- **Расширяемые панели** - по клику/выделению показывают дополнительный контент
- **Плавные анимации** - корутины для изменения размеров с кастомизируемой кривой

### Визуальные эффекты
- **Кастомный шейдер градиента** - 4-цветные градиенты для UI элементов
- **Поддержка Mask** - корректная работа с компонентами маскирования
- **Производительная графика** - оптимизированные шейдеры без потери качества
